 Blockchain Assignments

A collection of Blockchain and Web3 assignments implemented using Python, Solidity, Ethereum, and Web3 technologies.  
This repository covers fundamental blockchain concepts such as cryptography, smart contracts, mining, wallets, decentralized applications (DApps), token transfer systems, and blockchain security.
