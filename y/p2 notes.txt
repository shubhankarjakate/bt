# Practical 2: Implement Block Verification Function to Validate Chain Integrity

This practical is EXTREMELY important because this is the actual security idea behind blockchain.

---

# 1. 🧠 Intuition First

## 📖 Real-World Analogy

Imagine a teacher keeps answer sheets in a chain.

Each answer sheet contains:

* its own student data
* signature of previous sheet

```text id="d6n2wl"
Sheet1 → Sheet2 → Sheet3
```

Now if someone secretly changes marks in Sheet1:

* Sheet1 signature changes
* Sheet2 notices mismatch
* entire chain becomes suspicious

This checking process is called:

# ✅ Blockchain Verification

---

# 🔥 Main Idea

Blockchain integrity means:

```text id="s6g3y0"
No block data should be secretly modified
```

Verification function checks:

* Is hash correct?
* Is previous hash matching?
* Is chain still connected?

---

# 2. ⚙️ Technical Explanation

---

# 🔗 What Must Be Verified?

For every block:

| Check                  | Purpose                |
| ---------------------- | ---------------------- |
| Current hash valid?    | Detect block tampering |
| Previous hash matches? | Detect broken chain    |

---

# 🧱 Verification Logic

Suppose:

```text id="dzc1xh"
Block2.previous_hash == Block1.hash
```

If TRUE:
✅ chain valid

If FALSE:
❌ chain tampered

---

# 🔍 Two Important Checks

---

# ✅ Check 1: Current Hash Validation

Recalculate block hash again.

If:

```text id="o8iy9x"
stored_hash != recalculated_hash
```

means:

* block data changed
* blockchain compromised

---

# ✅ Check 2: Previous Hash Validation

Verify:

```text id="1hkh0g"
current.previous_hash == previous.hash
```

If mismatch:

* block chain broken

---

# 📘 Exam-Ready Definition

> Blockchain verification is the process of validating block hashes and previous hash links to ensure integrity and detect tampering in the blockchain.

---

# 4. 💻 Practical / Project Relevance

Used in:

* Bitcoin node validation
* Ethereum verification
* Transaction validation
* Detecting fraud
* Blockchain auditing

Without verification:

* blockchain security fails

# 🔄 Verification Flow

```text id="vh3yl6"
Check Current Hash

        ↓
Blockchain Valid
```

# 👨‍💻 Full Python Code with Verification

```python id="xjlwm5"
import hashlib
import datetime


# Block Class
class Block:

    def __init__(self, index, data, previous_hash):

        self.index = index
        self.timestamp = str(datetime.datetime.now())
        self.data = data
        self.previous_hash = previous_hash

        self.hash = self.calculate_hash()

    # Calculate hash
    def calculate_hash(self):

        block_data = (
            str(self.index) +
            self.timestamp +
            self.data +
            self.previous_hash
        )

        return hashlib.sha256(block_data.encode()).hexdigest()


# Blockchain Class
class Blockchain:

    def __init__(self):

        self.chain = [self.create_genesis_block()]

    # Create Genesis Block
    def create_genesis_block(self):

        return Block(0, "Genesis Block", "0")

    # Get latest block
    def get_latest_block(self):

        return self.chain[-1]

    # Add new block
    def add_block(self, data):

        previous_block = self.get_latest_block()

        new_block = Block(
            len(self.chain),
            data,
            previous_block.hash
        )

        self.chain.append(new_block)

    # Verify blockchain integrity
    def is_chain_valid(self):

        # Start from second block
        for i in range(1, len(self.chain)):

            current_block = self.chain[i]
            previous_block = self.chain[i - 1]

            # Check current hash
            if current_block.hash != current_block.calculate_hash():

                print("Current hash mismatch at block", i)
                return False

            # Check previous hash link
            if current_block.previous_hash != previous_block.hash:

                print("Previous hash mismatch at block", i)
                return False

        return True


# Create blockchain
my_blockchain = Blockchain()

# Add blocks
my_blockchain.add_block("Alice sends 10 BTC to Bob")
my_blockchain.add_block("Bob sends 5 BTC to Charlie")
my_blockchain.add_block("Charlie sends 2 BTC to David")


# Check validity
print("Blockchain Valid:", my_blockchain.is_chain_valid())


# Tamper block data
my_blockchain.chain[1].data = "Alice sends 1000 BTC to Hacker"


# Check validity again
print("Blockchain Valid After Tampering:",
      my_blockchain.is_chain_valid())
```
