Your `calculate_hash()` function is probably outside the `Block` class.

# ✅ Correct Code (FULL WORKING VERSION)

Copy this entire code freshly and run it.

```python id="t6e73w"
import hashlib
import datetime


# Block Class
class Block:

    def __init__(self, index, data, previous_hash):

        self.index = index
        self.timestamp = str(datetime.datetime.now())
        self.data = data
        self.previous_hash = previous_hash

        # Generate hash
        self.hash = self.calculate_hash()

    # Function to calculate hash
    def calculate_hash(self):

        block_data = (
            str(self.index) +
            self.timestamp +
            self.data +
            self.previous_hash
        )

        return hashlib.sha256(block_data.encode()).hexdigest()


# Blockchain Class
class Blockchain:

    def __init__(self):

        self.chain = [self.create_genesis_block()]

    # Create first block
    def create_genesis_block(self):

        return Block(0, "Genesis Block", "0")

    # Get latest block
    def get_latest_block(self):

        return self.chain[-1]

    # Add new block
    def add_block(self, data):

        previous_block = self.get_latest_block()

        new_block = Block(
            len(self.chain),
            data,
            previous_block.hash
        )

        self.chain.append(new_block)


# Create blockchain object
my_blockchain = Blockchain()

# Add blocks
my_blockchain.add_block("Alice sends 10 BTC to Bob")
my_blockchain.add_block("Bob sends 5 BTC to Charlie")
my_blockchain.add_block("Charlie sends 2 BTC to David")

# Print blockchain
for block in my_blockchain.chain:

    print("\n---------------------")
    print("Block Index:", block.index)
    print("Timestamp:", block.timestamp)
    print("Data:", block.data)
    print("Previous Hash:", block.previous_hash)
    print("Hash:", block.hash)
```

# 🔍 VERY IMPORTANT — Check Indentation

Python depends heavily on indentation.

This:

```python id="4d3vlh"
class Block:
```

Everything inside class MUST be indented.

Example:

```python id="n4n3zf"
class Block:

    def calculate_hash(self):
```

Notice spaces before `def`.
